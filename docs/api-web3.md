# Web3 API

```js
import Web3 from '@erebos/api-web3'
```

### new Web3()

Creates a Web3 instance using the provided `rpc` handler.

**Arguments**

1.  `rpc: RPC`

### .clientVersion()

**Returns** `Promise<string>`

### .sha3()

**Arguments**

1.  `data: hex`

**Returns** `Promise<hex>`
